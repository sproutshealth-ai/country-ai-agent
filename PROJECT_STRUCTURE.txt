🚀 QUICK START

1. Install dependencies:
   pip install -r requirements.txt

2. Set up API key:
   cp .env.example .env
   # Edit .env and add your OPENAI_API_KEY

3. Run the app:
   python app.py

4. Open browser:
   http://localhost:8000
